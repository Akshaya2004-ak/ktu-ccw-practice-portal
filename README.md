# 📚 KTU CCW Practice Portal

## About The Project

The **KTU CCW Practice Portal** is an MCQ practice platform designed for **KTU ECE students** preparing for the **Comprehensive Written (CCW) Examination**.

## Features

- 📖 Subject-wise practice (Analog, LCD, LIC, DSP, ADC)
- 🎯 Full mock tests (50 questions in 60 minutes)
- ⏱️ Countdown timer for mock tests
- ✅ Instant feedback (correct/wrong answers)
- 📊 Subject-wise performance analytics



## Live Demo

🔗 https://ktu-ccw-practice.streamlit.app/**



## How to Use

### Subject Practice
1. Click on any subject card
2. Enter number of questions
3. Answer and click "Check Answer"
4. View your score

### Mock Test
1. Click "ATTEMPT FULL MOCK TEST"
2. Complete 50 questions within 60 minutes
3. Submit to see subject-wise results
